# FIFA World Cup 2026 Lab

Project workspace for data, experiments, models, and tooling around the 2026 World Cup.

## Layout

- **data/** — raw, processed, and external datasets
- **notebooks/** — exploratory analysis and experiments
- **src/** — reusable library code (data, features, models, embeddings, simulation, API, dashboard)
- **configs/** — configuration files
- **tests/** — automated tests
- **models/** — trained artifacts and exports
- **reports/** — generated reports and figures
